第四章 电路定理

本章介绍一系列重要的电路定理，它们是线性电路分析的核心工具，可以大大简化复杂电路的分析过程。

§4-1 叠加定理

定理内容：在线性电阻电路中，任一处的电压或电流响应，等于各个独立电源单独作用时在该处产生的响应的代数和。
核心概念：
线性性质：包含齐次性（激励放大K倍，响应也放大K倍）和可加性（多个激励产生的总响应等于各激励单独产生响应之和）。叠加定理是可加性的直接体现。
“单独作用”的含义：当一个独立电源作用时，其他独立电源必须“置零”。
电压源置零：用短路代替。
电流源置零：用开路代替。
受控源的处理：受控源不是激励，必须保留在每个分电路中。
公式解释：
若电路中有电压源 \(u_{s1}, u_{s2}, ...\) 和电流源 \(i_{s1}, i_{s2}, ...\)，则任一响应 \(y\) 可表示为：
\[y = k_1 u_{s1} + k_2 u_{s2} + \dots + k_1' i_{s1} + k_2' i_{s2} + \dots\]
其中 \(k_1, k_2, k_1', k_2'\) 是由电路结构和元件参数决定的常数。
重要注意事项：
1. 只适用于线性电路，不适用于非线性电路。
2. 功率不能叠加。因为功率是电压或电流的二次函数，\(P = ui = (u'+u'')(i'+i'') \ne u'i' + u''i''\)。
3. 叠加时要注意各分量的参考方向，代数和中的符号由分量与原响应参考方向是否一致决定。
齐性定理：当所有激励同时增大或缩小 \(K\) 倍时，响应也增大或缩小 \(K\) 倍。特别地，当电路只有一个激励时，响应与该激励成正比。此定理常用于分析梯形电路。

§4-2 替代定理

定理内容：在任意集总电路中，如果某支路的电压 \(u_k\) 或电流 \(i_k\) 已知，则该支路可以用一个电压为 \(u_k\) 的理想电压源或电流为 \(i_k\) 的理想电流源替代，替代后电路中其他部分的电压和电流保持不变。
适用条件：
1. 被替代的支路可以是线性的，也可以是非线性的。
2. 替代前和替代后，电路必须有唯一解。
应用场景：在理论推导中很有用（如证明戴维宁定理），也可以用于简化局部电路。

§4-3 戴维宁定理和诺顿定理

核心思想：将任何一个含独立电源、线性电阻和线性受控源的一端口网络，对外等效为一个简单的有源支路。
戴维宁定理（等效电压源定理）：
内容：一个含源一端口，对外可以等效为一个理想电压源 \(u_{oc}\) 与一个电阻 \(R_{eq}\) 的串联组合。
参数确定：
开路电压 \(u_{oc}\)：端口开路时，端口处的电压。
等效电阻 \(R_{eq}\)：将一端口内所有独立电源置零（电压源短路，电流源开路）后，从端口看进去的输入电阻。
诺顿定理（等效电流源定理）：
内容：一个含源一端口，对外可以等效为一个理想电流源 \(i_{sc}\) 与一个电阻 \(R_{eq}\) 的并联组合。
参数确定：
短路电流 \(i_{sc}\)：端口短路时，流过短路线的电流。
等效电阻 \(R_{eq}\)：与戴维宁定理中的 \(R_{eq}\) 是同一个电阻。
重要关系：
\[u_{oc} = R_{eq} i_{sc}\]
等效电阻 \(R_{eq}\) 的求法：
1. 串并联法：适用于纯电阻网络。
2. 外加电源法：将独立电源置零后，在端口外加电压源 \(u\)，求端口电流 \(i\)，则 \(R_{eq} = u/i\)。
3. 开路短路法：同时求出开路电压 \(u_{oc}\) 和短路电流 \(i_{sc}\)，则 \(R_{eq} = u_{oc} / i_{sc}\)。此方法特别适用于含受控源的电路。

§4-4 最大功率传输定理

问题：一个含源一端口，当负载 \(R_L\) 变化时，何时能从该一端口获得最大功率？
定理内容：当负载电阻 \(R_L\) 等于含源一端口的等效电阻 \(R_{eq}\) 时（即 \(R_L = R_{eq}\)），负载能获得最大功率。
最大功率计算公式：
\[P_{Lmax} = \frac{u_{oc}^2}{4R_{eq}} = \frac{i_{sc}^2 R_{eq}}{4}\]
应用背景：在通信、电子电路中，追求最大功率传输（效率不高）。在电力系统中，追求高传输效率，通常负载电阻远大于电源内阻。

§4-5 特勒根定理

定理1（功率守恒）：对于一个具有 \(b\) 条支路的集总电路，若各支路电压 \(u_k\) 和电流 \(i_k\) 取关联参考方向，则所有支路吸收的功率之和恒为零。
\[\sum_{k=1}^{b} u_k i_k = 0\]
解释：这是功率守恒的数学表达，即“发出的功率 = 吸收的功率”。
定理2（拟功率定理）：对于两个具有相同拓扑结构（图相同）但元件不同的电路 \(N\) 和 \(\hat{N}\)，有：
\[\sum_{k=1}^{b} u_k \hat{i}_k = 0 \quad \text{和} \quad \sum_{k=1}^{b} \hat{u}_k i_k = 0\]
解释：一个电路的支路电压与另一个电路的支路电流的乘积之和为零。它没有直接的物理意义，但为证明互易定理等提供了数学工具。

§4-6 互易定理

适用条件：只适用于仅含线性电阻（不含受控源、独立源、耦合电感等）的单一激励电路。
核心内容：激励和响应互换位置后，响应与激励的比值保持不变。共有三种形式：
1. 形式一：电压源激励 → 短路电流响应。互换后，电流源激励 → 开路电压响应，且 \(\frac{i_2}{u_s} = \frac{\hat{u}_1}{\hat{i}_s}\)。当激励大小相等时，响应大小也相等。
2. 形式二：电流源激励 → 开路电压响应。互换后，电压源激励 → 短路电流响应。
3. 形式三：电压源激励 → 开路电压响应。互换后，电流源激励 → 短路电流响应。
实质：互易性是线性网络的一种性质，体现了电路在能量传递上的双向性。

§4-7 对偶原理

原理内容：如果两个电路（或方程、定理）的元素和结构存在一一对应的对偶关系，那么其中一个的结论可以通过互换对偶元素得到另一个的结论。
常见对偶元素：
电压 \(u\) ↔ 电流 \(i\)
电阻 \(R\) ↔ 电导 \(G\)
电感 \(L\) ↔ 电容 \(C\)
KVL ↔ KCL
串联 ↔ 并联
开路 ↔ 短路
戴维宁定理 ↔ 诺顿定理
应用：利用对偶性可以成倍地理解和记忆电路知识。

第五章 含有运算放大器的电阻电路

本章介绍运算放大器（运放）这一重要的多端元件，并分析含有理想运放的电阻电路。

§5-1 运算放大器的电路模型

运算放大器（运放）：一种高增益、高输入电阻、低输出电阻的直接耦合放大器。
电路符号：三角形符号，有两个输入端和一个输出端。
反相输入端（-）：电压 \(u^-\)。输出电压与该端输入电压相位相反。
同相输入端（+）：电压 \(u^+\)。输出电压与该端输入电压相位相同。
差分输入电压：\(u_d = u^+ - u^-\)。
输出特性：\(u_o = A u_d = A (u^+ - u^-)\)，其中 \(A\) 是开环电压放大倍数（很大，通常 \(10^4 \sim 10^7\)）。
运放的线性工作区：输出电压 \(u_o\) 与输入电压 \(u_d\) 成正比。由于 \(A\) 很大，线性区对应的输入 \(u_d\) 非常小。
运放的饱和特性：输出电压受电源电压限制，存在正、负饱和值 \(U_{sat}\)。当 \(u_o = U_{sat}\) 或 \(u_o = -U_{sat}\) 时，运放工作在非线性区。

§5-2 理想运算放大器

理想化条件：为了简化分析，将实际运放理想化，假设：
1. 开环增益 \(A \to \infty\)。
2. 输入电阻 \(R_i \to \infty\)。
3. 输出电阻 \(R_o \to 0\)。
理想运放的两个重要规则（在线性工作区内）：
1. “虚断”：由于 \(R_i \to \infty\)，流入两个输入端的电流为零，即 \(i^+ = i^- = 0\)。
2. “虚短”：由于 \(u_o = A (u^+ - u^-)\) 为有限值，且 \(A \to \infty\)，则必须 \(u^+ - u^- = 0\)，即 \(u^+ = u^-\)。

§5-3 含有理想运放的电路分析

分析方法：利用“虚断”和“虚短”两条规则，结合结点电压法或KCL、KVL进行求解。
典型电路：
1. 倒向比例器（反相比例放大器）：
电路结构：输入信号 \(u_i\) 通过电阻 \(R_1\) 接入反相输入端，同相输入端接地。反馈电阻 \(R_f\) 连接在输出端和反相输入端之间。
输出电压：
\[u_o = -\frac{R_f}{R_1} u_i\]
解释：输出与输入成比例，比例系数由电阻比决定，相位相反（故称“倒向”）。
2. 同相比例器：
电路结构：输入信号 \(u_i\) 接入同相输入端，反相输入端通过电阻 \(R_1\) 接地，并通过反馈电阻 \(R_f\) 接到输出端。
输出电压：
\[u_o = \left(1 + \frac{R_f}{R_1}\right) u_i\]
解释：输出与输入同相，且增益大于等于1。
3. 电压跟随器：
电路结构：同相比例器中令 \(R_f = 0\)（短路）或 \(R_1 = \infty\)（开路）得到。
输出电压：\(u_o = u_i\)。
特点：输入电阻极高，输出电阻极低，常用于隔离或缓冲，以消除负载效应。
4. 加法器：
电路结构：多个输入信号通过各自的输入电阻加到反相输入端。
输出电压：
\[u_o = -\left( \frac{R_f}{R_1} u_1 + \frac{R_f}{R_2} u_2 + \dots \right)\]
解释：输出是各输入电压按权值相加后的反相。

第六章 储能元件

本章介绍电容和电感这两种能够储存能量的元件，它们是构成动态电路的基础。

§6-1 电容元件

定义：能够储存电场能量的理想二端元件。
电路符号：两条平行的线段。
特性方程（库伏特性）：\(q = Cu\)
解释：电荷 \(q\)（单位：库仑，C）与电压 \(u\)（单位：伏特，V）成正比，比例系数 \(C\) 称为电容，单位是法拉（F）。
电压电流关系（VCR）：\(i = C \frac{du}{dt}\)
解释：电容电流与电压的变化率成正比。电压变化越快，电流越大。
直流特性：在直流电路中，\(\frac{du}{dt}=0\)，所以 \(i=0\)。因此，电容在直流稳态下相当于开路（“隔直”作用）。
记忆特性：\(u(t) = \frac{1}{C} \int_{-\infty}^{t} i(\xi) d\xi = u(t_0) + \frac{1}{C} \int_{t_0}^{t} i(\xi) d\xi\)
解释：电容电压不仅取决于当前电流，还与过去的充电历史有关，因此电容是记忆元件。
储能公式：\(W_C(t) = \frac{1}{2} C u^2(t)\)
解释：电容储存的电场能量与电压的平方成正比。电容在充电时吸收能量，放电时释放能量，本身不消耗能量，是无源元件和储能元件。

§6-2 电感元件

定义：能够储存磁场能量的理想二端元件。
电路符号：一组半圆形弧线。
特性方程（韦安特性）：\(\Psi = Li\)
解释：磁通链 \(\Psi\)（单位：韦伯，Wb）与电流 \(i\)（单位：安培，A）成正比，比例系数 \(L\) 称为电感，单位是亨利（H）。
电压电流关系（VCR）：\(u = L \frac{di}{dt}\)
解释：电感电压与电流的变化率成正比。电流变化越快，电压越大。
直流特性：在直流电路中，\(\frac{di}{dt}=0\)，所以 \(u=0\)。因此，电感在直流稳态下相当于短路。
记忆特性：\(i(t) = \frac{1}{L} \int_{-\infty}^{t} u(\xi) d\xi = i(t_0) + \frac{1}{L} \int_{t_0}^{t} u(\xi) d\xi\)
解释：电感电流不仅取决于当前电压，还与过去的“电压历史”有关，因此电感也是记忆元件。
储能公式：\(W_L(t) = \frac{1}{2} L i^2(t)\)
解释：电感储存的磁场能量与电流的平方成正比。电感在电流增加时吸收能量，电流减小时释放能量，本身不消耗能量，是无源元件和储能元件。

§6-3 电容、电感元件的串联与并联

电容串联：
等效电容：\(\frac{1}{C_{eq}} = \frac{1}{C_1} + \frac{1}{C_2} + \dots + \frac{1}{C_n}\)
解释：等效电容的倒数等于各电容倒数的和。这类似于电阻的并联，等效电容小于任一个串联电容。
电容并联：
等效电容：\(C_{eq} = C_1 + C_2 + \dots + C_n\)
解释：等效电容等于各电容之和。这类似于电阻的串联，等效电容大于任一个并联电容。
电感串联：
等效电感：\(L_{eq} = L_1 + L_2 + \dots + L_n\)
解释：等效电感等于各电感之和。这类似于电阻的串联。
电感并联：
等效电感：\(\frac{1}{L_{eq}} = \frac{1}{L_1} + \frac{1}{L_2} + \dots + \frac{1}{L_n}\)
解释：等效电感的倒数等于各电感倒数的和。这类似于电阻的并联，等效电感小于任一个并联电感。
初始值的等效：当电容或电感的初始电压或初始电流不为零时，进行串并联等效时，需要将初始值按基尔霍夫定律进行等效合并。

第七章 一阶电路和二阶电路的时域分析

本章研究动态电路在换路后，电压、电流随时间变化的规律，核心是建立并求解微分方程。这种直接在时间域内分析的方法称为**时域分析法**。

§7-1 动态电路的方程及其初始条件

动态电路：含有动态元件（电容、电感）的电路。其方程是微分方程，因为电容的 \(i = C du/dt\) 和电感的 \(u = L di/dt\) 定义了导数关系。
换路：电路结构或参数发生的突变（如开关闭合/断开、元件参数突然改变）。换路是引起过渡过程的原因。
换路定则：在换路瞬间（\(t=0\)），如果电容电流和电感电压为有限值，则：
电容电压不能跃变：\(u_C(0_+) = u_C(0_-)\)
电感电流不能跃变：\(i_L(0_+) = i_L(0_-)\)
解释：电容电压跃变需要无穷大的电流（\(i = C du/dt\)），电感电流跃变需要无穷大的电压（\(u = L di/dt\)），这在物理上不可能实现。因此，换路定则是分析初始条件的根本依据。
初始条件（初始值）：响应及其各阶导数在 \(t=0_+\) 时刻的值。
独立初始条件：\(u_C(0_+)\) 和 \(i_L(0_+)\)。它们由换路定则直接从换路前的稳态值 \(u_C(0_-)\) 和 \(i_L(0_-)\) 得到。
非独立初始条件：其他支路的电压、电流在 \(t=0_+\) 的值。求法：画出 \(t=0_+\) 的等效电路。
电容的处理：用电压为 \(u_C(0_+)\) 的电压源替代。
电感的处理：用电流为 \(i_L(0_+)\) 的电流源替代。
若初始值为零，电容相当于短路（电压源为0），电感相当于开路（电流源为0）。

§7-2 一阶电路的零输入响应

定义：**零输入响应**是电路在没有外施激励（独立电源置零），仅由动态元件初始储能所产生的响应。
RC电路的零输入响应：
电路描述：一个已充电的电容 \(C\)（初始电压 \(U_0\)）通过电阻 \(R\) 放电。
微分方程：由KVL得 \(u_R + u_C = 0\)，代入 \(u_R = iR\)，\(i = -C du_C/dt\)（注意放电电流方向），得到：
\[RC \frac{du_C}{dt} + u_C = 0\]
方程的解：这是一个一阶齐次微分方程。设解为 \(u_C = Ae^{pt}\)，代入得特征方程 \(RCp + 1 = 0\)，特征根 \(p = -1/(RC)\)。利用初始条件 \(u_C(0_+) = U_0\)，解得：
\[u_C(t) = U_0 e^{-\frac{t}{RC}} \quad (t \ge 0)\]
电流：\(i(t) = -C \frac{du_C}{dt} = \frac{U_0}{R} e^{-\frac{t}{RC}}\)
时间常数 \(\tau = RC\)：
定义：\(\tau\) 是响应衰减到初始值的 \(1/e \approx 36.8\%\) 所需的时间。
物理意义：\(\tau\) 衡量过渡过程进行的快慢。\(\tau\) 越大，衰减越慢，过渡过程越长。
RL电路的零输入响应：
电路描述：一个初始电流为 \(I_0\) 的电感 \(L\) 通过电阻 \(R\) 释放能量。
微分方程：\(L \frac{di}{dt} + Ri = 0\)
方程的解：特征根 \(p = -R/L\)，利用 \(i(0_+) = I_0\)，得：
\[i_L(t) = I_0 e^{-\frac{R}{L}t}\]
电感电压：\(u_L(t) = L \frac{di}{dt} = -R I_0 e^{-\frac{R}{L}t}\)
时间常数 \(\tau = L/R\)：物理意义与RC电路相同，决定了RL电路过渡过程的快慢。

§7-3 一阶电路的零状态响应

定义：**零状态响应**是电路在动态元件初始储能为零的条件下，仅由外施激励产生的响应。
RC电路的零状态响应（直流激励）：
电路描述：初始电压为0的电容 \(C\) 通过电阻 \(R\) 被直流电压源 \(U_s\) 充电。
微分方程：\(RC \frac{du_C}{dt} + u_C = U_s\)
方程的解：这是一个一阶非齐次微分方程。解由特解（强制分量）和通解（自由分量）组成。
特解：\(u_{Cp} = U_s\)（稳态时的电容电压）。
通解：\(u_{Ch} = Ae^{-t/(RC)}\)。
代入初始条件 \(u_C(0_+) = 0\)，得 \(A = -U_s\)。因此：
\[u_C(t) = U_s (1 - e^{-\frac{t}{RC}})\]
充电电流：\(i(t) = C \frac{du_C}{dt} = \frac{U_s}{R} e^{-\frac{t}{RC}}\)
RL电路的零状态响应（直流激励）：
电路描述：初始电流为0的电感 \(L\) 与电阻 \(R\) 串联，接直流电流源 \(I_s\)。
微分方程：\(\frac{L}{R} \frac{di_L}{dt} + i_L = I_s\)
方程的解：类似地，得：
\[i_L(t) = I_s (1 - e^{-\frac{R}{L}t})\]
正弦激励下的零状态响应：
核心结论：全响应 = 强制分量（与激励同频率的正弦稳态分量）+ 自由分量（随时间衰减的指数项）。
合闸角的影响：自由分量的大小与开关闭合的时刻（即激励的初相 \(\phi_u\)）有关。当 \(\phi_u - \phi_z = \pm 90^\circ\) 时，自由分量最大，可能导致电路中出现过电流（接近稳态值的两倍）。

§7-4 一阶电路的全响应

定义：**全响应**是电路在非零初始状态和外施激励共同作用下产生的响应。由叠加定理，全响应 = 零输入响应 + 零状态响应。
三要素法：适用于直流激励的一阶电路。
公式：
\[f(t) = f(\infty) + [f(0_+) - f(\infty)] e^{-\frac{t}{\tau}}\]
三要素详解：
1. 初始值 \(f(0_+)\)：换路后 \(t=0_+\) 时响应 \(f(t)\) 的值。利用换路定则和 \(0_+\) 等效电路求解。
2. 稳态值 \(f(\infty)\)：换路后电路达到新稳态时 \(f(t)\) 的值。在直流稳态下，电容视为开路，电感视为短路。
3. 时间常数 \(\tau\)：
对于RC电路：\(\tau = R_{eq} C\)。
对于RL电路：\(\tau = L / R_{eq}\)。
其中 \(R_{eq}\) 是换路后从动态元件两端看进去的戴维宁等效电阻。
物理意义：三要素法将求解微分方程的过程简化为三个关键参数的确定，是分析一阶电路最核心的方法。

§7-5 二阶电路的零输入响应

RLC串联放电电路：初始电压为 \(U_0\) 的电容 \(C\) 通过 \(R\)、\(L\) 放电。
特征方程：由微分方程 \(LC \frac{d^2u_C}{dt^2} + RC \frac{du_C}{dt} + u_C = 0\) 设解 \(u_C = Ae^{pt}\)，得：
\[LC p^2 + RC p + 1 = 0\]
特征根：
\[p_{1,2} = -\frac{R}{2L} \pm \sqrt{\left(\frac{R}{2L}\right)^2 - \frac{1}{LC}} = -\delta \pm \sqrt{\delta^2 - \omega_0^2}\]
\(\delta = R/(2L)\)：**衰减常数**，决定振荡衰减的快慢。
\(\omega_0 = 1/\sqrt{LC}\)：**无阻尼谐振角频率**，决定振荡的快慢。
三种放电过程：
1. 过阻尼（非振荡放电）：\(R > 2\sqrt{L/C}\)（即 \(\delta > \omega_0\)）。
特征根：两个不等的负实数，\(p_1 \ne p_2 < 0\)。
响应形式：\(u_C(t) = A_1 e^{p_1 t} + A_2 e^{p_2 t}\)，单调衰减至零，不振荡。
2. 临界阻尼：\(R = 2\sqrt{L/C}\)（即 \(\delta = \omega_0\)）。
特征根：相等的负实数，\(p_1 = p_2 = -\delta\)。
响应形式：\(u_C(t) = (A_1 + A_2 t) e^{-\delta t}\)，处于振荡与非振荡的临界状态，响应最快地衰减到零。
3. 欠阻尼（振荡放电）：\(R < 2\sqrt{L/C}\)（即 \(\delta < \omega_0\)）。
特征根：一对共轭复数，\(p_{1,2} = -\delta \pm j\omega_d\)，其中 \(\omega_d = \sqrt{\omega_0^2 - \delta^2}\) 称为**衰减振荡角频率**。
响应形式：\(u_C(t) = \frac{U_0 \omega_0}{\omega_d} e^{-\delta t} \sin(\omega_d t + \beta)\)，呈衰减振荡。
当 \(R=0\)（无损耗）时，\(\delta=0\)，\(\omega_d=\omega_0\)，响应为等幅振荡 \(u_C(t) = U_0 \cos(\omega_0 t)\)。

§7-6 二阶电路的零状态响应和全响应

分析方法：与一阶电路类似，全响应 = 特解（强制分量）+ 通解（自由分量）。自由分量的形式（过阻尼、欠阻尼等）由特征根决定。
关键区别：二阶电路需要**两个初始条件**（通常是 \(u_C(0_+)\) 和 \(i_L(0_+)\)）来确定通解中的两个积分常数。

§7-7 一阶电路和二阶电路的阶跃响应

单位阶跃函数 \(\epsilon(t)\)：
\[\epsilon(t) = \begin{cases} 0, & t<0 \\ 1, & t>0 \end{cases}\]
定义：在 \(t=0\) 时刻从0跳变到1的函数。
物理意义：它是理想化地描述开关在 \(t=0\) 时刻接通单位直流源（1V或1A）的数学模型。
阶跃响应 \(s(t)\)：电路在单位阶跃激励 \(\epsilon(t)\) 作用下的零状态响应。
延迟阶跃函数 \(\epsilon(t-t_0)\)：表示在 \(t=t_0\) 时刻发生跳变。可用于表示矩形脉冲：\(\epsilon(t) - \epsilon(t-t_0)\)。

§7-8 一阶电路和二阶电路的冲激响应

单位冲激函数 \(\delta(t)\)：
定义：\(\delta(t) = 0 \ (t \ne 0)\)，且 \(\int_{-\infty}^{\infty} \delta(t) dt = 1\)。
物理意义：宽度趋于0、幅度趋于无穷大、但面积为1的脉冲。它是一个广义函数，不是一个常规函数。
重要性质：
\(\delta(t) = \frac{d\epsilon(t)}{dt}\)（冲激是阶跃的导数）。
筛分性质：\(\int_{-\infty}^{\infty} f(t) \delta(t-t_0) dt = f(t_0)\)。
冲激响应 \(h(t)\)：电路在单位冲激激励 \(\delta(t)\) 作用下的零状态响应。
重要关系：\(h(t) = \frac{ds(t)}{dt}\)，\(s(t) = \int_{-\infty}^{t} h(\tau) d\tau\)。即**冲激响应是阶跃响应的导数**。
物理效应：单位冲激电压源能瞬间改变电感电流，单位冲激电流源能瞬间改变电容电压。之后，电路在非零初始状态下进行零输入响应。

§7-9 卷积积分

定义：两个函数 \(f_1(t)\) 和 \(f_2(t)\) 的卷积为：
\[f_1(t) * f_2(t) = \int_{0}^{t} f_1(t-\tau) f_2(\tau) d\tau\]
它满足交换律、分配律和结合律。
核心应用：对于线性时不变电路，任意激励 \(e(t)\) 下的零状态响应 \(r(t)\)，等于激励 \(e(t)\) 与单位冲激响应 \(h(t)\) 的卷积：
\[r(t) = e(t) * h(t) = \int_{0}^{t} e(\tau) h(t-\tau) d\tau\]
解释：将任意激励 \(e(t)\) 分解为一系列强度为 \(e(\tau)d\tau\) 的冲激函数。电路的响应就是这些冲激响应 \(h(t-\tau)\) 的叠加（积分）。卷积积分是连接时域与频域（拉普拉斯变换、傅里叶变换）的桥梁。

第八章 相量法

本章引入复数表示正弦量，将时域的微积分方程转化为频域的代数方程，是分析正弦稳态电路的基础。

§8-1 复数

复数的表示：
代数形式：\(F = a + jb\)，其中 \(j = \sqrt{-1}\) 为虚数单位。
三角形式：\(F = |F|(\cos\theta + j\sin\theta)\)。
指数形式/极坐标形式：\(F = |F|e^{j\theta} = |F| \angle \theta\)（工程常用）。
其中，**模** \(|F| = \sqrt{a^2+b^2}\)，**辐角** \(\theta = \arctan(b/a)\)。
复数的运算：
加减：使用代数形式，实部与实部、虚部与虚部分别相加减。
乘除：使用极坐标形式更方便。模相乘除，辐角相加减。
旋转因子：\(e^{j\theta}\)。一个复数乘以 \(e^{j\theta}\) 相当于在复平面上将其逆时针旋转角度 \(\theta\)。特别地，\(j = e^{j90^\circ}\)，乘以 \(j\) 相当于逆时针旋转 \(90^\circ\)。

§8-2 正弦量

正弦量：按正弦规律变化的电压或电流，统称为正弦量。
正弦量的三要素（以电流为例）：\(i(t) = I_m \cos(\omega t + \phi_i)\)
1. 振幅 \(I_m\)：正弦量的最大值，单位A（安培）。
2. 角频率 \(\omega\)：相位变化的角速度，\(\omega = 2\pi f = \frac{2\pi}{T}\)，单位rad/s。
3. 初相 \(\phi_i\)：\(t=0\) 时的相位，通常取主值范围 \(|\phi_i| \le 180^\circ\)。
相位差：两个同频正弦量的相位之差，\(\phi = \phi_u - \phi_i\)。它是一个与时间无关的常数。
\(\phi > 0\)：电压**超前**电流（电压先达到正峰值）。
\(\phi < 0\)：电压**滞后**电流。
\(\phi = 0\)：**同相**。
\(\phi = \pm 90^\circ\)：**正交**。
\(\phi = \pm 180^\circ\)：**反相**。
有效值：
定义：周期量 \(i(t)\) 的有效值 \(I\) 是它的**均方根值**：\(I = \sqrt{\frac{1}{T} \int_0^T i^2(t) dt}\)。
物理意义：一个周期电流的有效值在数值上等于产生相同热效应的直流电流值。
正弦量的有效值：\(I = \frac{I_m}{\sqrt{2}} \approx 0.707 I_m\)。类似地，\(U = U_m/\sqrt{2}\)。

§8-3 相量法的基础

相量的定义：对于一个正弦量 \(i(t) = \sqrt{2} I \cos(\omega t + \phi_i)\)，其对应的**有效值相量**定义为：
\[\dot{I} = I \angle \phi_i = I e^{j\phi_i}\]
核心思想：建立一个从时域正弦函数到频域复数的**一一对应关系**。相量 \(\dot{I}\) 完整地包含了正弦量的**有效值 \(I\)** 和**初相 \(\phi_i\)**，但隐含了角频率 \(\omega\)。
注意：相量不是正弦量，而是一个复数。两者是“对应”关系，不是“相等”关系。
相量运算规则：
微分运算：\(\frac{di(t)}{dt} \longleftrightarrow j\omega \dot{I}\)。时域的微分对应频域中乘以 \(j\omega\)。
积分运算：\(\int i(t) dt \longleftrightarrow \frac{1}{j\omega} \dot{I}\)。时域的积分对应频域中除以 \(j\omega\)。

§8-4 电路定律的相量形式

KCL的相量形式：\(\sum \dot{I} = 0\)。流出任一结点的电流相量代数和为零。
KVL的相量形式：\(\sum \dot{U} = 0\)。沿任一回路绕行的电压相量代数和为零。
电阻VCR的相量形式：\(\dot{U}_R = R \dot{I}_R\)。
解释：电阻的电压相量与电流相量**同相位**。其模值满足欧姆定律 \(U_R = R I_R\)。
电感VCR的相量形式：\(\dot{U}_L = j\omega L \dot{I}_L = jX_L \dot{I}_L\)。
解释：电感的电压相量**超前**电流相量 \(90^\circ\)。\(X_L = \omega L\) 称为**感抗**，单位 \(\Omega\)，它表示电感对电流的阻碍作用，且与频率成正比。
电容VCR的相量形式：\(\dot{U}_C = \frac{1}{j\omega C} \dot{I}_C = -j\frac{1}{\omega C} \dot{I}_C = -jX_C \dot{I}_C\)。
解释：电容的电压相量**滞后**电流相量 \(90^\circ\)。\(X_C = 1/(\omega C)\) 称为**容抗**，单位 \(\Omega\)，它表示电容对电流的阻碍作用，且与频率成反比。

第九章 正弦稳态电路的分析

本章应用相量法，将电阻电路的分析方法系统地推广到正弦稳态电路中。

§9-1 阻抗和导纳

复阻抗 \(Z\)（定义）：无源一端口网络端口电压相量与电流相量之比。
\[Z = \frac{\dot{U}}{\dot{I}} = |Z| \angle \phi_z = R + jX\]
阻抗模 \(|Z|\)：\(|Z| = U/I\)，单位 \(\Omega\)。表示端口电压有效值与电流有效值之比。
阻抗角 \(\phi_z\)：\(\phi_z = \phi_u - \phi_i\)。表示端口电压与电流的相位差。
电阻分量 \(R\)：\(R = |Z| \cos \phi_z\)，单位 \(\Omega\)。
电抗分量 \(X\)：\(X = |Z| \sin \phi_z\)，单位 \(\Omega\)。当 \(X > 0\) 时，\(Z\) 呈感性；当 \(X < 0\) 时，\(Z\) 呈容性。
复导纳 \(Y\)（定义）：无源一端口网络端口电流相量与电压相量之比。
\[Y = \frac{\dot{I}}{\dot{U}} = |Y| \angle \phi_y = G + jB\]
导纳模 \(|Y|\)：\(|Y| = I/U = 1/|Z|\)，单位 S。
导纳角 \(\phi_y\)：\(\phi_y = \phi_i - \phi_u = -\phi_z\)。
电导分量 \(G\)：\(G = |Y| \cos \phi_y\)，单位 S。
电纳分量 \(B\)：\(B = |Y| \sin \phi_y\)，单位 S。当 \(B > 0\) 时，\(Y\) 呈容性；当 \(B < 0\) 时，\(Y\) 呈感性。
重要关系：\(Z = 1/Y\)。
串并联等效互换：一个串联电路 \((R, X)\) 可以等效为一个并联电路 \((G', B')\)，反之亦然。
\[G' = \frac{R}{R^2+X^2}, \quad B' = -\frac{X}{R^2+X^2}\] \[R = \frac{G'}{G'^2+B'^2}, \quad X = -\frac{B'}{G'^2+B'^2}\]

§9-2 正弦稳态电路的分析

核心方法：**相量变换**。
1. 将时域电路中的正弦量替换为对应的相量。
2. 将电阻 \(R\)、电感 \(L\)、电容 \(C\) 替换为对应的复阻抗 \(Z_R = R\)、\(Z_L = j\omega L\)、\(Z_C = 1/(j\omega C)\)。
3. 此时，电路变成了一个**相量模型**，它是一个与电阻电路结构完全相同的复数电路。
4. 在相量模型中，所有电阻电路的分析方法（支路法、网孔法、结点法、叠加定理、戴维宁定理等）和公式（如分压、分流公式）都**直接适用**。
5. 求解得到未知电压、电流的相量，最后反变换回时域正弦函数。

§9-3 正弦稳态电路的功率

瞬时功率 \(p(t)\)：\(p(t) = u(t)i(t)\)。对于正弦稳态电路：
\[p(t) = UI\cos\phi_z [1 + \cos(2\omega t)] - UI\sin\phi_z \sin(2\omega t)\]
第一项：**不可逆部分**，始终 \(\ge 0\)，其平均值即为有功功率。
第二项：**可逆部分**，以 \(2\omega\) 振荡，平均值为零，代表能量在电源与储能元件之间来回交换。
有功功率（平均功率）\(P\)（定义）：瞬时功率在一个周期内的平均值。
\[P = UI\cos\phi_z = I^2 R = \frac{U^2}{R}\]
物理意义：电阻消耗的功率，是电路实际做功的功率，单位瓦特（W）。\(\cos\phi_z\) 称为**功率因数**。
无功功率 \(Q\)（定义）：
\[Q = UI\sin\phi_z = I^2 X = \frac{U^2}{X}\]
物理意义：衡量储能元件（L、C）与电源之间能量交换的**最大速率**，单位乏（var）。
感性负载（\(Q>0\)）吸收无功功率，容性负载（\(Q<0\)）发出无功功率。
视在功率 \(S\)（定义）：
\[S = UI = \sqrt{P^2 + Q^2}\]
物理意义：电源提供的总功率容量，单位伏安（VA）。电气设备的额定容量通常用视在功率表示。
功率三角形：\(P, Q, S\) 构成一个直角三角形，\(S\) 为斜边，\(P\) 为邻边，\(Q\) 为对边，\(\phi_z\) 为功率因数角。

§9-4 复功率

定义：为了统一表示三种功率，引入**复功率** \(\overline{S}\)：
\[\overline{S} = \dot{U} \dot{I}^* = P + jQ\]
其中 \(\dot{I}^*\) 是电流相量 \(\dot{I}\) 的共轭复数。
模 \(|\overline{S}| = S\)（视在功率）。
实部 \(\Re(\overline{S}) = P\)（有功功率）。
虚部 \(\Im(\overline{S}) = Q\)（无功功率）。
辐角 \(\arg(\overline{S}) = \phi_z\)（功率因数角）。
重要性质：复功率守恒。对于任何电路，所有支路吸收的复功率之和为零：\(\sum \overline{S} = 0\)。这等价于有功功率守恒（\(\sum P=0\)）和无功功率守恒（\(\sum Q=0\)）。

§9-5 最大功率传输

问题描述：一个含源一端口（戴维宁等效参数为 \(\dot{U}_{oc}\) 和 \(Z_{eq} = R_{eq} + jX_{eq}\)）向一个可变负载 \(Z_L = R_L + jX_L\) 传输有功功率。求 \(Z_L\) 为何值时能获得最大有功功率。
共轭匹配条件：
\[Z_L = Z_{eq}^* = R_{eq} - jX_{eq}\]
解释：负载的电阻应等于电源内阻的电阻；负载的电抗应与电源内阻的电抗**数值相等、性质相反**（即感性负载匹配容性内阻，或反之）。这样，电路中的电抗被完全抵消，电路呈纯阻性。
最大有功功率公式：
\[P_{Lmax} = \frac{U_{oc}^2}{4R_{eq}}\]
注意：此时传输效率仅为50%。此定理适用于通信、电子等追求功率最大化的弱电领域，不适用于追求高效率的电力系统。

第十章 含有耦合电感的电路

本章研究当两个或多个线圈之间存在磁耦合时，电路的分析方法。核心是理解“互感”的物理本质以及如何用“同名端”来正确处理互感电压的极性。

§10-1 互感——核心重难点

一、基本概念
自感：一个线圈自身的电流产生的磁通与该线圈自身交链的现象。
磁通链：\(\Psi_{11} = L_1 i_1\)，\(\Psi_{22} = L_2 i_2\)
\(L_1, L_2\)：自感系数，单位亨利（H）。它只取决于线圈本身的匝数、形状、尺寸和介质。
互感：一个线圈的电流产生的磁通与另一个线圈交链的现象。
磁通链：\(\Psi_{12} = M_{12} i_2\)（线圈2的电流在线圈1中产生的磁通链）
\(\Psi_{21} = M_{21} i_1\)（线圈1的电流在线圈2中产生的磁通链）
重要定理：\(M_{12} = M_{21} = M\)。互感系数 \(M\) 是对称的，单位也是亨利（H）。
耦合电感的磁通链方程：
\[\Psi_1 = \Psi_{11} \pm \Psi_{12} = L_1 i_1 \pm M i_2\] \[\Psi_2 = \Psi_{21} \pm \Psi_{22} = \pm M i_1 + L_2 i_2\]
“±”号的物理意义：取决于两个线圈的磁场是相互增强还是相互削弱。
“+”号：磁场相互增强（同向耦合）
“-”号：磁场相互削弱（反向耦合）

二、同名端——处理互感极性的关键

为什么需要同名端？
因为实际线圈往往被封装起来，无法直观看到绕向。要判断互感电压的极性，需要一个简化的标记方法。**同名端**就是为了解决这个问题而引入的概念。
定义：当两个线圈的电流分别从两个端子流入（或流出）时，若它们产生的磁场相互增强（即互感起增磁作用），则这两个端子称为**同名端**（或同极性端），通常用“·”或“*”标记。
同名端的物理意义：
同名端是互感极性的一种标志。它表明：**当施感电流从同名端流入时，该电流产生的互感电压在该线圈上的“+”极性端也在同名端**。

三、耦合电感的VCR

时域形式：
\[u_1 = L_1 \frac{di_1}{dt} \pm M \frac{di_2}{dt}\] \[u_2 = \pm M \frac{di_1}{dt} + L_2 \frac{di_2}{dt}\]
取号规则（重要！）：
若两电流均从同名端流入（或均从同名端流出），则互感电压项取“+”号。
若一个从同名端流入，另一个从异名端流入，则互感电压项取“-”号。
相量形式（正弦稳态）：
\[\dot{U}_1 = j\omega L_1 \dot{I}_1 \pm j\omega M \dot{I}_2\] \[\dot{U}_2 = \pm j\omega M \dot{I}_1 + j\omega L_2 \dot{I}_2\]

四、耦合因数 \(k\)

\[k = \frac{M}{\sqrt{L_1 L_2}} \le 1\]
定义：衡量两个线圈耦合紧密程度的物理量。
物理意义：
\(k=1\)：**全耦合**。一个线圈产生的磁通全部与另一个线圈交链，没有漏磁通。
\(k=0\)：无耦合。两个线圈之间没有磁联系。

§10-2 含有耦合电感电路的计算——重难点

一、串联
同向串联：两线圈的同名端相连。
等效电感：\(L_{eq} = L_1 + L_2 + 2M\)
反向串联：两线圈的异名端相连。
等效电感：\(L_{eq} = L_1 + L_2 - 2M\)
工程应用：通过测量两种串联方式下的等效电感，可以计算出互感\(M\)：
\[M = \frac{L_{同} - L_{反}}{4}\]

二、去耦等效法（T形去耦）——最重要方法！

适用条件：两个耦合电感有一条支路相连（即构成T形或π形结构）。
核心思想：用一个无耦合的等效电路替代原耦合电感电路，使计算大大简化。
等效规则：
\[ \begin{cases} L_{1'} = L_1 \mp M \\ L_{2'} = L_2 \mp M \\ L_{3'} = \pm M \end{cases} \]
取号规则：
同侧连接（同名端在公共结点）：取“-”号。
异侧连接（异名端在公共结点）：取“+”号。

§10-3 耦合电感的功率

关键结论：
1. 互感不消耗有功功率，它只是能量传递的通道。
2. 互感的无功功率可呈现感性或容性特性。

§10-4 变压器原理

一次侧等效电路——反映阻抗概念（核心！）
\[Z_i = Z_{11} + \frac{(\omega M)^2}{Z_{22}}\]
反映阻抗：二次侧的阻抗通过互感反映到一次侧，性质与原阻抗相反。

§10-5 理想变压器

理想化条件：无损耗、全耦合、电感无穷大。
端口特性方程：
\[u_1 = n u_2, \quad i_1 = -\frac{1}{n} i_2\]
阻抗变换特性：
\[Z_i = n^2 Z_L\]
结论：理想变压器不消耗、不储存能量，只进行电压、电流、阻抗变换。

第十一章 电路的频率响应

本章研究电路响应随输入信号频率变化的规律，即**频率响应**，是滤波、选频、谐振电路的理论基础。

§11-1 频率响应的基本概念

定义：电路稳态响应相量 \(\dot{R}\) 与激励相量 \(\dot{E}\) 之比随频率 \(\omega\) 变化的函数关系，称为**网络函数** \(H(j\omega)\)。
\[H(j\omega) = \frac{\dot{R}}{\dot{E}} = |H(\omega)| \angle \varphi(\omega)\]
\(|H(\omega)|\)：幅频特性（增益/衰减随频率变化）
\(\varphi(\omega)\)：相频特性（相位随频率变化）

§11-2 RC电路的频率响应

1. RC低通电路
输出取自电容电压，允许低频信号通过，抑制高频信号。
截止角频率：\(\omega_c = \frac{1}{RC}\)
2. RC高通电路
输出取自电阻电压，允许高频信号通过，抑制低频信号。
截止角频率：\(\omega_c = \frac{1}{RC}\)

§11-3 串联谐振电路

谐振条件：\(X_L = X_C \implies \omega L = \frac{1}{\omega C}\)
谐振角频率：\(\omega_0 = \frac{1}{\sqrt{LC}}\)
串联谐振特点：
1. 阻抗最小且为纯电阻 \(Z=R\)，电流最大
2. 电感与电容电压大小相等、相位相反，可能远大于电源电压（电压谐振）
3. 电路总功率因数为1，呈纯阻性
品质因数 \(Q\)：
\[Q = \frac{\omega_0 L}{R} = \frac{1}{\omega_0 C R} = \frac{1}{R}\sqrt{\frac{L}{C}}\]
Q值越高，谐振曲线越尖锐，选频能力越强。

§11-4 并联谐振电路

谐振条件：\(B=0\)（总电纳为零）
谐振角频率：\(\omega_0 \approx \frac{1}{\sqrt{LC}}\)（高Q近似）
并联谐振特点：
1. 阻抗最大且为纯电阻
2. 端口电流最小，电感与电容支路电流近似相等（电流谐振）
3. 功率因数为1

§11-5 波特图

波特图是频率响应的对数坐标图，分为幅频特性和相频特性。
幅频：纵坐标20lg|H|（dB），横坐标lgω
相频：纵坐标相位，横坐标lgω
用途：直观观察电路在宽频率范围内的增益与相位变化。

第十二章 三相电路

本章介绍电力系统的基石——三相制。核心是理解对称三相电路的计算方法、线值与相值关系、功率计算及中性线作用。

§12-1 三相电路的基本概念

对称三相电源：
\[u_A = \sqrt{2}U_p \cos(\omega t)\] \[u_B = \sqrt{2}U_p \cos(\omega t - 120^\circ)\] \[u_C = \sqrt{2}U_p \cos(\omega t + 120^\circ)\]
相量和：\(\dot{U}_A + \dot{U}_B + \dot{U}_C = 0\)
相序：正序A→B→C（电力系统标准）、负序、零序
连接方式：星形（Y）、三角形（Δ）

§12-2 线电压(电流)与相电压(电流)的关系

星形连接（Y）：
线电流 = 相电流 \(I_l = I_p\)
线电压 = \(\sqrt{3}\) 相电压，超前对应相电压30° \(U_l = \sqrt{3}U_p\)
三角形连接（Δ）：
线电压 = 相电压 \(U_l = U_p\)
线电流 = \(\sqrt{3}\) 相电流，滞后对应相电流30° \(I_l = \sqrt{3}I_p\)

§12-3 对称三相电路的计算

核心方法：归结为一相法
原理：对称Y-Y电路中性点电位相等 \(\dot{U}_{N'N}=0\)
步骤：
1. 所有Δ负载/电源变换为Y型 \(Z_Y=Z_\Delta/3\)
2. 画出一相（A相）等效电路
3. 计算A相电流电压
4. 由对称性直接写出B、C相结果

§12-4 不对称三相电路

中性点位移：\(\dot{U}_{N'N} \ne 0\)，导致各相电压不对称
中性线的作用：强制中性点电位相等，保证负载相电压对称
安全规则：中性线严禁接保险丝或开关，防止断开造成设备损坏
中性线电流：\(\dot{I}_N = \dot{I}_A + \dot{I}_B + \dot{I}_C \ne 0\)

§12-5 三相电路的功率

对称三相电路功率：
有功功率：\(P = 3U_pI_p\cos\phi = \sqrt{3}U_lI_l\cos\phi\)
无功功率：\(Q = 3U_pI_p\sin\phi = \sqrt{3}U_lI_l\sin\phi\)
视在功率：\(S = 3U_pI_p = \sqrt{3}U_lI_l\)
注意：\(\phi\) 是**相电压与相电流**的相位差
瞬时功率：恒定值，是三相电机运行平稳的原因
二瓦计法：适用于三相三线制，总功率 \(P=P_1+P_2\)

第十三章 非正弦周期电流电路和信号的频谱

本章研究当激励信号不是正弦波时，线性电路的分析方法。核心思想是谐波分析法：将非正弦周期信号分解为一系列正弦波的叠加。

§13-1 非正弦周期信号

什么是非正弦周期信号？ 波形不是正弦曲线，但仍然按周期重复变化的信号。
常见例子：矩形波（方波）：数字电路中广泛使用；三角波：扫描电路；锯齿波：示波器扫描；全波整流波形：电源电路。
为什么需要研究？ 实际信号大多不是纯净的正弦波。即使电源是正弦的，电路中的非线性元件（如二极管、三极管）也会产生非正弦的电压和电流。通信、控制、计算机等领域大量使用非正弦信号（脉冲、方波等）。
基本思路：将非正弦周期信号分解为一系列频率成整数倍的正弦波的叠加。分别计算每个正弦波单独作用下的响应（利用正弦稳态分析法）。将各响应叠加，得到总响应。

§13-2 非正弦周期函数分解为傅里叶级数

一、傅里叶级数的定义 任何一个满足狄里赫利条件的周期函数 \(f(t)\)（周期为 \(T\)，角频率 \(\omega_1 = 2\pi/T\)）都可以展开为傅里叶级数：
三角形式：\(f(t) = \frac{a_0}{2} + \sum_{k=1}^{\infty} [a_k \cos(k\omega_1 t) + b_k \sin(k\omega_1 t)]\)
余弦形式（更常用）：\(f(t) = \frac{A_0}{2} + \sum_{k=1}^{\infty} A_{km} \cos(k\omega_1 t + \phi_k)\)
二、各项的物理意义
项名称角频率物理意义
\(\frac{A_0}{2}\)直流分量（恒定分量）0信号的平均值
\(k=1\)基波分量\(\omega_1\)与原信号同频率的主要成分
\(k=2,3,4,\dots\)高次谐波\(2\omega_1, 3\omega_1, 4\omega_1, \dots\)整数倍频率的成分
谐波次数越高，频率越高，波形细节越丰富。
三、系数计算公式 \(a_k = \frac{2}{T} \int_0^T f(t) \cos(k\omega_1 t) dt,\quad b_k = \frac{2}{T} \int_0^T f(t) \sin(k\omega_1 t) dt,\quad A_{km} = \sqrt{a_k^2 + b_k^2}, \quad \phi_k = \arctan\left(-\frac{b_k}{a_k}\right)\)。\(a_0 = \frac{2}{T} \int_0^T f(t) dt\)，是直流分量的两倍（所以直流分量为 \(a_0/2\)）。
四、利用对称性简化计算
对称性条件系数特征
偶函数\(f(t)=f(-t)\)\(b_k=0\)，只有余弦项
奇函数\(f(t)=-f(-t)\)\(a_k=0\)，只有正弦项
奇谐波函数\(f(t)=-f(t+T/2)\)只有奇次谐波（\(k\)为奇数）
技巧：合理选择计时起点（即坐标原点），可以简化傅里叶级数。
五、典型波形的傅里叶级数 矩形波（方波，对称于原点）：\(f(t) = \frac{4E_m}{\pi} \left( \sin \omega_1 t + \frac{1}{3} \sin 3\omega_1 t + \frac{1}{5} \sin 5\omega_1 t + \cdots \right)\)。特点：只含奇次正弦谐波，谐波幅度与次数成反比衰减。

§13-3 有效值、平均值和平均功率

一、有效值（均方根值） 对于非正弦周期电流 \(i(t) = I_0 + \sum_{k=1}^{\infty} I_{km} \cos(k\omega_1 t + \phi_k)\)：\(I = \sqrt{I_0^2 + I_1^2 + I_2^2 + I_3^2 + \cdots} = \sqrt{I_0^2 + \sum_{k=1}^{\infty} I_k^2}\)。核心结论：非正弦周期量的有效值等于直流分量和各次谐波有效值的平方和开根。注意：这里 \(I_k = I_{km}/\sqrt{2}\) 是各次谐波的有效值。
二、平均值 \(I_{av} = \frac{1}{T} \int_0^T |i(t)| dt\)。与有效值的区别：有效值是平方平均，平均值是绝对值的平均。正弦波：\(I_{av} = \frac{2\sqrt{2}}{\pi} I \approx 0.637 I\)，\(I = 1.11 I_{av}\)（波形因数）。
三、平均功率（有功功率） \(P = U_0 I_0 + \sum_{k=1}^{\infty} U_k I_k \cos \phi_k\)。核心结论：不同频率的电压和电流不产生平均功率，只有同频率的电压和电流才贡献平均功率。解释：\(\int_0^T \cos(m\omega_1 t) \cos(n\omega_1 t) dt = 0\) 当 \(m \ne n\)。

§13-4 非正弦周期电流电路的计算——谐波分析法

计算步骤：1. 傅里叶分解：将非正弦周期激励分解为直流分量和各次谐波。2. 分别计算：直流分量：电容开路，电感短路，按直流电阻电路计算。各次谐波：用相量法计算。注意：感抗 \(X_L(k) = k\omega_1 L\)，容抗 \(X_C(k) = -1/(k\omega_1 C)\)，与谐波次数 \(k\) 有关。3. 叠加：将各分量计算结果在时域叠加（不能直接叠加相量，因为频率不同！）。正确做法：将各次谐波的瞬时表达式写出，然后相加。
关键注意事项：谐振现象：对于某次谐波，电路可能发生谐振（如 \(k\omega_1 L = 1/(k\omega_1 C)\)），该次谐波响应会很大。收敛性：实际计算中只能取有限项。波形越光滑，级数收敛越快。功率计算：总功率等于各次谐波功率之和（包括直流）。

§13-5 傅里叶级数的指数形式

利用欧拉公式 \(e^{j\theta} = \cos\theta + j\sin\theta\)，可将傅里叶级数写成：\(f(t) = \sum_{k=-\infty}^{\infty} c_k e^{jk\omega_1 t}\)，其中 \(c_k = \frac{1}{T} \int_0^T f(t) e^{-jk\omega_1 t} dt\)。特点：\(k\) 取正负整数，频谱对称（\(|c_{-k}| = |c_k|\)）。优点：便于与傅里叶变换衔接。

§13-6 频谱

定义：以频率（或角频率）为横坐标，以各次谐波的振幅（或 \(c_k\) 的模）为纵坐标画出的线状图形。特点：离散性：只在基波频率的整数倍处有谱线。收敛性：谐波次数越高，幅度越小。带宽：信号能量集中的频率范围。

第十四章 线性动态电路的复频域分析

本章引入拉普拉斯变换，将时域的微分方程转化为频域的代数方程，是求解高阶动态电路的有力工具。

§14-1 拉普拉斯变换的定义

一、为什么需要拉普拉斯变换？ 时域法的困难：对于高阶电路，求解微分方程需要确定多个积分常数，计算繁琐。拉氏变换的优势：将微分方程转化为代数方程；自动包含初始条件，无需单独求积分常数；可以处理冲激函数等奇异函数。
二、定义 对于定义在 \([0, \infty)\) 的时间函数 \(f(t)\)，其拉普拉斯变换（简称拉氏变换）定义为：\(F(s) = \mathcal{L}[f(t)] = \int_{0_-}^{\infty} f(t) e^{-st} dt\)。其中\(s = \sigma + j\omega\)：复频率，是复数变量；\(F(s)\)：象函数，是 \(s\) 的复变函数；\(f(t)\)：原函数。积分下限取 \(0_-\)：可以包含 \(t=0\) 时刻的冲激函数。
三、常用函数的拉氏变换
原函数 \(f(t)\)象函数 \(F(s)\)
单位冲激函数 \(\delta(t)\)1
单位阶跃函数 \(\epsilon(t)\)\(1/s\)
指数函数 \(e^{-at}\)\(1/(s+a)\)
正弦函数 \(\sin(\omega t)\)\(\omega/(s^2+\omega^2)\)
余弦函数 \(\cos(\omega t)\)\(s/(s^2+\omega^2)\)
幂函数 \(t^n\)\(n!/s^{n+1}\)

§14-2 拉普拉斯变换的基本性质

一、线性性质 \(\mathcal{L}[af_1(t) + bf_2(t)] = aF_1(s) + bF_2(s)\)
二、微分性质（最重要！） \(\mathcal{L}\left[\frac{df(t)}{dt}\right] = sF(s) - f(0_-)\)，\(\mathcal{L}\left[\frac{d^2f(t)}{dt^2}\right] = s^2F(s) - sf(0_-) - f'(0_-)\)。意义：时域的微分运算，在频域中变为乘以 \(s\) 并减去初始值。初始条件被自动计入！
三、积分性质 \(\mathcal{L}\left[\int_{0_-}^t f(\xi) d\xi\right] = \frac{F(s)}{s}\)
四、延迟性质 \(\mathcal{L}[f(t-t_0)\epsilon(t-t_0)] = e^{-st_0}F(s)\)。意义：时域延迟 \(t_0\)，频域乘以 \(e^{-st_0}\)。

§14-3 拉普拉斯反变换——部分分式展开法

一、一般形式 电路的响应象函数通常是 \(s\) 的有理分式：\(F(s) = \frac{N(s)}{D(s)} = \frac{a_m s^m + a_{m-1}s^{m-1} + \cdots + a_0}{b_n s^n + b_{n-1}s^{n-1} + \cdots + b_0} \quad (n > m)\)
二、部分分式展开的三种情况
情况1：\(D(s)=0\) 有单根 设 \(p_1, p_2, \dots, p_n\) 为单根，则 \(F(s) = \frac{K_1}{s-p_1} + \frac{K_2}{s-p_2} + \cdots + \frac{K_n}{s-p_n}\)，\(K_i = (s-p_i)F(s)\big|_{s=p_i} = \frac{N(p_i)}{D'(p_i)}\)，原函数 \(f(t) = \sum_{i=1}^n K_i e^{p_i t}\)。
情况2：\(D(s)=0\) 有共轭复根 设 \(p_{1,2} = -\alpha \pm j\omega\)，则对应的两项可合并为：\(\mathcal{L}^{-1}\left[\frac{K}{s+\alpha-j\omega} + \frac{K^*}{s+\alpha+j\omega}\right] = 2|K|e^{-\alpha t}\cos(\omega t + \angle K)\)。
情况3：\(D(s)=0\) 有重根 设 \(s=p_1\) 为 \(q\) 重根，则 \(F(s) = \frac{K_{1q}}{(s-p_1)^q} + \frac{K_{1(q-1)}}{(s-p_1)^{q-1}} + \cdots + \frac{K_{11}}{s-p_1} + \cdots\)。系数计算公式（以 \(q=3\) 为例）：\(K_{11} = (s-p_1)^3 F(s)\big|_{s=p_1}\)，\(K_{12} = \frac{d}{ds}[(s-p_1)^3 F(s)]\big|_{s=p_1}\)，\(K_{13} = \frac{1}{2}\frac{d^2}{ds^2}[(s-p_1)^3 F(s)]\big|_{s=p_1}\)。

§14-4 运算电路

一、基尔霍夫定律的运算形式 KCL：\(\sum I(s) = 0\)；KVL：\(\sum U(s) = 0\)。
二、电路元件的运算模型 电阻：\(U(s) = R I(s)\)。电感：\(U(s) = sL I(s) - L i(0_-)\)，等效电路：电感 \(L\) 与电压源 \(L i(0_-)\) 串联（方向与电流参考方向相反）。电容：\(U(s) = \frac{1}{sC} I(s) + \frac{u_C(0_-)}{s}\)，等效电路：电容 \(C\) 与电压源 \(u_C(0_-)/s\) 串联。互感：\(U_1(s) = sL_1 I_1(s) - L_1 i_1(0_-) + sM I_2(s) - M i_2(0_-)\)，\(U_2(s) = sL_2 I_2(s) - L_2 i_2(0_-) + sM I_1(s) - M i_1(0_-)\)。
三、运算阻抗 定义：在零初始条件下，元件两端的电压象函数与电流象函数之比。电阻：\(Z_R(s) = R\)；电感：\(Z_L(s) = sL\)；电容：\(Z_C(s) = 1/(sC)\)。运算形式的欧姆定律：\(U(s) = Z(s) I(s)\)（零初始条件下）。

§14-5 应用拉普拉斯变换法分析线性电路

分析步骤：1. 画运算电路：将时域电路中的电压源、电流源用它们的象函数表示；将电阻、电感、电容用它们的运算阻抗表示；将初始条件用附加电源表示（电感串联电压源，电容串联电压源）。2. 列方程：在运算电路中，用电阻电路的分析方法（结点法、网孔法、戴维宁定理等）列出代数方程。3. 求解象函数：解方程，得到响应的象函数 \(R(s)\)。4. 拉氏反变换：将 \(R(s)\) 进行部分分式展开，查表求原函数 \(r(t)\)。
优点：自动计入初始条件；将微分方程转化为代数方程；可以方便地处理冲激函数。

§14-6 网络函数

定义：在线性时不变电路中，在零初始条件下，响应象函数与激励象函数之比：\(H(s) = \frac{R(s)}{E(s)}\)。
物理意义：当 \(E(s)=1\)（即 \(e(t)=\delta(t)\)）时，\(R(s)=H(s)\)。因此，网络函数的原函数就是单位冲激响应：\(h(t) = \mathcal{L}^{-1}[H(s)]\)。
分类：驱动点函数：激励和响应在同一端口（阻抗 \(Z(s)\) 或导纳 \(Y(s)\)）。转移函数：激励和响应在不同端口。

§14-7 网络函数的极点和零点

标准形式：\(H(s) = H_0 \frac{(s-z_1)(s-z_2)\cdots(s-z_m)}{(s-p_1)(s-p_2)\cdots(s-p_n)}\)。零点：\(H(s)=0\) 的点，即分子多项式 \(N(s)=0\) 的根 \(z_1, z_2, \dots, z_m\)。极点：\(H(s) \to \infty\) 的点，即分母多项式 \(D(s)=0\) 的根 \(p_1, p_2, \dots, p_n\)。零极点图：在 \(s\) 平面上用“\(\circ\)”标零点，用“\(\times\)”标极点。
极点的性质：左半平面极点（\(\text{Re}[p_i] < 0\)）：对应衰减的瞬态响应，电路稳定。右半平面极点（\(\text{Re}[p_i] > 0\)）：对应增长的响应，电路不稳定。虚轴上的极点（\(\text{Re}[p_i] = 0\)）：对应等幅振荡（临界稳定）。

§14-8 极点、零点与冲激响应

\(h(t) = \sum_{i=1}^n K_i e^{p_i t}\)。极点决定响应的波形形态：负实极点 → 指数衰减；正实极点 → 指数增长（不稳定）；共轭复极点 → 衰减/增长振荡；虚轴共轭复极点 → 等幅振荡。极点位置与响应速度：极点离虚轴越远，衰减越快。

§14-9 极点、零点与频率响应

令 \(s = j\omega\)，则 \(H(j\omega) = H_0 \frac{(j\omega-z_1)(j\omega-z_2)\cdots}{(j\omega-p_1)(j\omega-p_2)\cdots}\)。幅频特性：\(|H(j\omega)| = |H_0| \frac{\prod |j\omega-z_i|}{\prod |j\omega-p_j|}\)；相频特性：\(\phi(\omega) = \sum \angle(j\omega-z_i) - \sum \angle(j\omega-p_j)\)。几何求法：在 \(s\) 平面上，\(|j\omega-z_i|\) 是点 \(j\omega\) 到零点 \(z_i\) 的距离，\(\angle(j\omega-z_i)\) 是该向量的辐角。

第十五章 电路方程的矩阵形式

本章介绍如何用矩阵形式系统地建立电路方程，为计算机辅助分析奠定基础。

§15-1 割集

一、割集的定义 割集 \(Q\) 是连通图 \(G\) 的一个支路集合，满足：1. 移去 \(Q\) 中的所有支路，图 \(G\) 被分成两个部分。2. 少移去其中任意一条支路，图 \(G\) 仍然是连通的。通俗理解：割集是把图“切断”所需的最少支路集合。判定方法：在图上作一个闭合面，与闭合面相交的支路构成一个割集。
二、基本割集（单树支割集） 选定一个树后，每一条树支与若干连支构成一个割集。这样的割集称为基本割集，共有 \(n-1\) 个（\(n\) 为结点数）。方向规定：基本割集的方向与该树支的方向一致。

§15-2 关联矩阵、回路矩阵、割集矩阵

一、关联矩阵 \(A\) 维度：\((n-1) \times b\)（降阶关联矩阵）。元素定义：\(a_{jk} = \begin{cases}+1 & \text{支路 }k \text{ 与结点 }j \text{ 关联，且方向背离结点} \\ -1 & \text{支路 }k \text{ 与结点 }j \text{ 关联，且方向指向结点} \\ 0 & \text{支路 }k \text{ 与结点 }j \text{ 无关联}\end{cases}\)。KCL的矩阵形式：\(A i = 0\)；KVL的矩阵形式：\(u = A^T u_n\)。
二、回路矩阵 \(B\) 维度：\(l \times b\)（\(l = b-n+1\) 为独立回路数）。元素定义：\(b_{jk} = \begin{cases}+1 & \text{支路 }k \text{ 与回路 }j \text{ 关联，且方向一致} \\ -1 & \text{支路 }k \text{ 与回路 }j \text{ 关联，且方向相反} \\ 0 & \text{支路 }k \text{ 与回路 }j \text{ 无关联}\end{cases}\)。基本回路矩阵 \(B_f\)：按“先连支、后树支”排列，有 \(B_f = [I_l \; B_t]\)。KVL的矩阵形式：\(B u = 0\)；KCL的矩阵形式：\(i = B^T i_l\)。
三、割集矩阵 \(Q\) 维度：\((n-1) \times b\)。元素定义：与关联矩阵类似，但行对应割集。基本割集矩阵 \(Q_f\)：按“先树支、后连支”排列，有 \(Q_f = [I_t \; Q_l]\)。KCL的矩阵形式：\(Q i = 0\)；KVL的矩阵形式：\(u = Q^T u_t\)。

§15-3 矩阵 \(A, B_f, Q_f\) 之间的关系

\(A B_f^T = 0\) 或 \(B_f A^T = 0\)；\(Q_f B_f^T = 0\) 或 \(B_f Q_f^T = 0\)；\(Q_l = -B_t^T = A_t^{-1} A_l\)。

§15-4 回路电流方程的矩阵形式

复合支路的定义：每条支路由阻抗 \(Z_k\)（或导纳 \(Y_k\)）、电压源 \(\dot{U}_{sk}\)、电流源 \(\dot{I}_{sk}\) 串联/并联组成。
支路方程的矩阵形式：\(\dot{U} = Z(\dot{I} + \dot{I}_s) - \dot{U}_s\) 或 \(\dot{I} = Y(\dot{U} + \dot{U}_s) - \dot{I}_s\)。
回路电流方程的推导：1. KCL：\(\dot{I} = B^T \dot{I}_l\)；2. KVL：\(B \dot{U} = 0\)；3. 支路方程：\(\dot{U} = Z \dot{I} + Z \dot{I}_s - \dot{U}_s\)。代入得：\(B Z B^T \dot{I}_l = B \dot{U}_s - B Z \dot{I}_s\)。回路阻抗矩阵 \(Z_l = B Z B^T\)：\(l \times l\) 方阵，主对角元为自阻抗，非对角元为互阻抗。

§15-5 结点电压方程的矩阵形式

结点电压方程的推导：1. KVL：\(\dot{U} = A^T \dot{U}_n\)；2. KCL：\(A \dot{I} = 0\)；3. 支路方程：\(\dot{I} = Y \dot{U} + Y \dot{U}_s - \dot{I}_s\)。代入得：\(A Y A^T \dot{U}_n = A \dot{I}_s - A Y \dot{U}_s\)。结点导纳矩阵 \(Y_n = A Y A^T\)：\((n-1) \times (n-1)\) 方阵。主对角元 \(Y_{ii}\)：与结点 \(i\) 相连的所有支路导纳之和（自导，正值）。非对角元 \(Y_{ij}\)：连接结点 \(i\) 和 \(j\) 的支路导纳的负值（互导，负值）。结点注入电流列向量 \(J_n = A \dot{I}_s - A Y \dot{U}_s\)。

§15-6 割集电压方程的矩阵形式

类似结点电压法，用基本割集矩阵 \(Q_f\) 代替关联矩阵 \(A\)：\(Q_f Y Q_f^T \dot{U}_t = Q_f \dot{I}_s - Q_f Y \dot{U}_s\)。割集导纳矩阵 \(Y_t = Q_f Y Q_f^T\)。物理意义：割集电压法可以看作是结点电压法的推广。当每个割集都汇集在一个结点上时，割集电压法就退化为结点电压法。

§15-7 列表法

为什么需要列表法？ 结点电压法：不允许无伴电压源。回路电流法：不允许无伴电流源。列表法：对支路类型无限制，适应性强。
基本思想：将支路电压 \(\dot{U}\)、支路电流 \(\dot{I}\)、结点电压 \(\dot{U}_n\) 都作为未知量，联立求解。
方程组成：1. KCL：\(A \dot{I} = 0\)（\(n-1\) 个方程）；2. KVL：\(\dot{U} - A^T \dot{U}_n = 0\)（\(b\) 个方程）；3. 支路方程：\(F \dot{U} + H \dot{I} = \dot{U}_s + \dot{I}_s\)（\(b\) 个方程）。
矩阵形式：\(\begin{bmatrix}0 & 0 & A \\ -A^T & I_b & 0 \\ 0 & F & H\end{bmatrix} \begin{bmatrix}\dot{U}_n \\ \dot{U} \\ \dot{I}\end{bmatrix} = \begin{bmatrix}0 \\ 0 \\ \dot{U}_s + \dot{I}_s\end{bmatrix}\)。总方程数：\(2b + n - 1\)。优点：适应性强，可以处理各种类型的支路（包括受控源、互感、理想变压器等）。缺点：方程数较多，但系数矩阵稀疏，适合计算机求解。

第十六章 二端口网络

本章研究具有两对端口的网络，分析其端口特性及参数描述。核心是理解各种参数的定义、物理意义及其相互转换。

§16-1 二端口网络

一、基本概念 - 二端口网络：具有两对端子的网络，且满足端口条件——从一个端子流入的电流等于从另一个端子流出的电流。- 与四端网络的区别：四端网络不要求满足端口条件（电流不一定成对相等）。- 常见应用：变压器、滤波器、放大器、反馈网络、传输线等。- 图形符号：一个方框，左边是端口1（输入），右边是端口2（输出）。
二、为什么要研究二端口？ - 黑箱思想：只关心端口的外部特性（电压、电流关系），不关心内部结构。- 模块化：复杂网络可以分解为多个简单二端口的连接。- 标准化：用统一的参数描述不同内部结构的网络，便于比较和设计。
三、基本约定 - 电压、电流参考方向：端口1：\(\dot{U}_1\) 和 \(\dot{I}_1\) 取关联方向（电流流入为正）。端口2：\(\dot{U}_2\) 和 \(\dot{I}_2\) 取关联方向（电流流入为正）。注意：输出端口电流 \(I_2\) 的参考方向通常指向网络内部（即负载电流流出网络）。

§16-2 二端口的方程和参数

一、Y参数（短路导纳参数） 定义方程：\(\begin{cases} \dot{I}_1 = Y_{11} \dot{U}_1 + Y_{12} \dot{U}_2 \\ \dot{I}_2 = Y_{21} \dot{U}_1 + Y_{22} \dot{U}_2 \end{cases}\)。参数含义：\(Y_{11} = \left. \frac{\dot{I}_1}{\dot{U}_1} \right|_{\dot{U}_2=0}\)（端口2短路时端口1输入导纳），\(Y_{12} = \left. \frac{\dot{I}_1}{\dot{U}_2} \right|_{\dot{U}_1=0}\)，\(Y_{21} = \left. \frac{\dot{I}_2}{\dot{U}_1} \right|_{\dot{U}_2=0}\)，\(Y_{22} = \left. \frac{\dot{I}_2}{\dot{U}_2} \right|_{\dot{U}_1=0}\)。矩阵形式：\(\begin{bmatrix} \dot{I}_1 \\ \dot{I}_2 \end{bmatrix} = \begin{bmatrix} Y_{11} & Y_{12} \\ Y_{21} & Y_{22} \end{bmatrix} \begin{bmatrix} \dot{U}_1 \\ \dot{U}_2 \end{bmatrix}\)。重要性质：互易性（对于仅含线性电阻、电感、电容的无源二端口，有 \(Y_{12}=Y_{21}\)，此时只有3个独立参数）；对称性（如果结构上对称，还有 \(Y_{11}=Y_{22}\)，此时只有2个独立参数）。
二、Z参数（开路阻抗参数） 定义方程：\(\begin{cases} \dot{U}_1 = Z_{11} \dot{I}_1 + Z_{12} \dot{I}_2 \\ \dot{U}_2 = Z_{21} \dot{I}_1 + Z_{22} \dot{I}_2 \end{cases}\)。参数含义：\(Z_{11} = \left. \frac{\dot{U}_1}{\dot{I}_1} \right|_{\dot{I}_2=0}\)（端口2开路时端口1输入阻抗），\(Z_{12} = \left. \frac{\dot{U}_1}{\dot{I}_2} \right|_{\dot{I}_1=0}\)，\(Z_{21} = \left. \frac{\dot{U}_2}{\dot{I}_1} \right|_{\dot{I}_2=0}\)，\(Z_{22} = \left. \frac{\dot{U}_2}{\dot{I}_2} \right|_{\dot{I}_1=0}\)。矩阵形式：\(\begin{bmatrix} \dot{U}_1 \\ \dot{U}_2 \end{bmatrix} = \begin{bmatrix} Z_{11} & Z_{12} \\ Z_{21} & Z_{22} \end{bmatrix} \begin{bmatrix} \dot{I}_1 \\ \dot{I}_2 \end{bmatrix}\)。Y与Z的关系：\(Z = Y^{-1}\)，转换公式：\(Z_{11} = \frac{Y_{22}}{\Delta_Y}, Z_{12} = -\frac{Y_{12}}{\Delta_Y}, Z_{21} = -\frac{Y_{21}}{\Delta_Y}, Z_{22} = \frac{Y_{11}}{\Delta_Y}\)，其中 \(\Delta_Y = Y_{11}Y_{22} - Y_{12}Y_{21}\)。
三、T参数（传输参数/A参数） 定义方程：\(\begin{cases} \dot{U}_1 = A \dot{U}_2 - B \dot{I}_2 \\ \dot{I}_1 = C \dot{U}_2 - D \dot{I}_2 \end{cases}\)。参数含义：\(A = \left. \frac{\dot{U}_1}{\dot{U}_2} \right|_{\dot{I}_2=0}\)（端口2开路时的电压传输比），\(B = \left. \frac{\dot{U}_1}{-\dot{I}_2} \right|_{\dot{U}_2=0}\)（端口2短路时的转移阻抗），\(C = \left. \frac{\dot{I}_1}{\dot{U}_2} \right|_{\dot{I}_2=0}\)（端口2开路时的转移导纳），\(D = \left. \frac{\dot{I}_1}{-\dot{I}_2} \right|_{\dot{U}_2=0}\)（端口2短路时的电流传输比）。矩阵形式：\(\begin{bmatrix} \dot{U}_1 \\ \dot{I}_1 \end{bmatrix} = \begin{bmatrix} A & B \\ C & D \end{bmatrix} \begin{bmatrix} \dot{U}_2 \\ -\dot{I}_2 \end{bmatrix}\)。重要性质：互易性（无源线性二端口满足 \(AD - BC = 1\)），对称性（如果结构对称，还有 \(A = D\)）。为什么T参数重要？ 最适合描述级联连接的二端口。
四、H参数（混合参数） 定义方程：\(\begin{cases} \dot{U}_1 = H_{11} \dot{I}_1 + H_{12} \dot{U}_2 \\ \dot{I}_2 = H_{21} \dot{I}_1 + H_{22} \dot{U}_2 \end{cases}\)。参数含义：\(H_{11} = \left. \frac{\dot{U}_1}{\dot{I}_1} \right|_{\dot{U}_2=0}\)（端口2短路时的输入阻抗），\(H_{12} = \left. \frac{\dot{U}_1}{\dot{U}_2} \right|_{\dot{I}_1=0}\)（端口1开路时的反向电压传输比），\(H_{21} = \left. \frac{\dot{I}_2}{\dot{I}_1} \right|_{\dot{U}_2=0}\)（端口2短路时的正向电流传输比），\(H_{22} = \left. \frac{\dot{I}_2}{\dot{U}_2} \right|_{\dot{I}_1=0}\)（端口1开路时的输出导纳）。应用：H参数广泛应用于晶体管的小信号建模（如 \(h_{fe}\)、\(h_{ie}\) 等）。

§16-3 二端口的等效电路

一、为什么需要等效电路？ 给定参数（Y、Z、T、H）后，可以用一个简单的等效电路替代原二端口。便于理解和分析二端口对外电路的影响。二、T形等效电路（对应Z参数） 对于无源线性二端口（\(Z_{12}=Z_{21}\)）：\(Z_1 = Z_{11} - Z_{12}, \quad Z_2 = Z_{12}, \quad Z_3 = Z_{22} - Z_{12}\)，电路结构：三个阻抗接成T形（星形）。三、π形等效电路（对应Y参数） 对于无源线性二端口（\(Y_{12}=Y_{21}\)）：\(Y_1 = Y_{11} + Y_{12}, \quad Y_2 = -Y_{12}, \quad Y_3 = Y_{22} + Y_{21}\)，电路结构：三个导纳接成π形（三角形）。四、含受控源的二端口等效电路 当二端口含有受控源时，\(Z_{12} \ne Z_{21}\) 或 \(Y_{12} \ne Y_{21}\)。此时等效电路中需包含受控源。例如，用Z参数表示时：\(\dot{U}_1 = Z_{11}\dot{I}_1 + Z_{12}\dot{I}_2\)，\(\dot{U}_2 = Z_{21}\dot{I}_1 + Z_{22}\dot{I}_2 = Z_{12}\dot{I}_1 + Z_{22}\dot{I}_2 + (Z_{21}-Z_{12})\dot{I}_1\)，这相当于在T形等效电路的基础上，在输出端串联一个CCVS。

§16-4 二端口的转移函数

定义：在零初始条件下，输出象函数与输入象函数之比。转移函数是网络函数在二端口中的具体应用。常见转移函数：
转移函数类型表达式条件
电压转移函数\(\frac{\dot{U}_2}{\dot{U}_1}\)\(\dot{I}_2=0\)（输出开路）
电流转移函数\(\frac{\dot{I}_2}{\dot{I}_1}\)\(\dot{U}_2=0\)（输出短路）
转移阻抗\(\frac{\dot{U}_2}{\dot{I}_1}\)\(\dot{I}_2=0\)
转移导纳\(\frac{\dot{I}_2}{\dot{U}_1}\)\(\dot{U}_2=0\)
用Z参数表示：\(\frac{\dot{U}_2}{\dot{U}_1} = \frac{Z_{21}}{Z_{11}}, \quad \frac{\dot{I}_2}{\dot{I}_1} = -\frac{Z_{21}}{Z_{22}}, \quad \frac{\dot{U}_2}{\dot{I}_1} = Z_{21}\)。用Y参数表示：\(\frac{\dot{U}_2}{\dot{U}_1} = -\frac{Y_{21}}{Y_{22}}, \quad \frac{\dot{I}_2}{\dot{I}_1} = \frac{Y_{21}}{Y_{11}}, \quad \frac{\dot{I}_2}{\dot{U}_1} = Y_{21}\)。

§16-5 二端口的连接

一、级联（链联） 连接方式：第一个二端口的输出端接第二个二端口的输入端。参数：T参数最方便。\(\begin{bmatrix} A & B \\ C & D \end{bmatrix} = \begin{bmatrix} A_1 & B_1 \\ C_1 & D_1 \end{bmatrix} \begin{bmatrix} A_2 & B_2 \\ C_2 & D_2 \end{bmatrix}\)。解释：级联时，总T参数矩阵等于各二端口T参数矩阵的乘积（按连接顺序）。二、并联 连接方式：两个二端口的输入端并联、输出端并联。条件：并联后各端口的端口条件不被破坏。参数：Y参数最方便。\(Y = Y_1 + Y_2\)。解释：并联时，总Y参数矩阵等于各二端口Y参数矩阵的和。三、串联 连接方式：两个二端口的输入端串联、输出端串联。条件：串联后各端口的端口条件不被破坏。参数：Z参数最方便。\(Z = Z_1 + Z_2\)。解释：串联时，总Z参数矩阵等于各二端口Z参数矩阵的和。

§16-6 回转器和负阻抗变换器

一、回转器 定义：一种线性非互易的二端口元件，其端口特性为：\(\begin{cases} u_1 = -r i_2 \\ u_2 = r i_1 \end{cases}\) 或 \(\begin{cases} i_1 = g u_2 \\ i_2 = -g u_1 \end{cases}\)，\(r\)：回转电阻，单位 \(\Omega\)；\(g = 1/r\)：回转电导，单位 S。重要特性：功率：\(u_1 i_1 + u_2 i_2 = 0\)，不消耗功率，是无源元件。非互易：\(Z_{12} \ne Z_{21}\)（\(Z_{12} = -r\)，\(Z_{21} = r\)）。核心应用——电感模拟：在输出端接电容 \(C\)，则输入端等效为电感：\(Z_{in} = \frac{u_1}{i_1} = \frac{-r i_2}{g u_2} = r^2 \frac{1}{sC} = s(r^2 C)\)，等效电感：\(L_{eq} = r^2 C\)。意义：在集成电路中，大电感难以制造，可以用回转器+电容来模拟电感。二、负阻抗变换器（NIC） 定义：一种二端口元件，可以将正阻抗变换为负阻抗。两种类型：1. 电流反向型：\(\begin{cases} U_1 = U_2 \\ I_1 = -k I_2 \end{cases}\)，输入阻抗：\(Z_{in} = \frac{U_1}{I_1} = \frac{U_2}{-k I_2} = -\frac{1}{k} Z_L\)。2. 电压反向型：\(\begin{cases} U_1 = -k U_2 \\ I_1 = I_2 \end{cases}\)，输入阻抗：\(Z_{in} = \frac{U_1}{I_1} = \frac{-k U_2}{I_2} = -k Z_L\)。应用：实现负电阻、负电感、负电容，用于振荡器、有源滤波器等。

第十七章 非线性电路

本章研究含有非线性元件的电路。核心是理解非线性元件的特性及分析方法。

§17-1 非线性电阻

一、基本概念 - 非线性电阻：伏安特性不是通过原点的直线，不满足欧姆定律。- 分类：
类型特性数学表达示例
电流控制型电压是电流的单值函数\(u = f(i)\)充气二极管
电压控制型电流是电压的单值函数\(i = g(u)\)隧道二极管
单调型同时满足上述两种\(u=f(i), i=g(u)\)PN结二极管
二、静态电阻与动态电阻 - 静态电阻：\(R = \frac{u}{i}\)，是工作点到原点的连线斜率。- 动态电阻：\(R_d = \frac{du}{di}\)，是工作点处切线的斜率。- 负电阻：当伏安特性曲线下倾时，动态电阻为负值。三、PN结二极管的伏安特性 \(i = I_s(e^{\frac{q}{kT}u} - 1)\)，\(I_s\)：反向饱和电流（很小，约 \(10^{-14}\)A量级），\(\frac{q}{kT} \approx 40 \, \text{V}^{-1}\)（室温下）。特点：正向电压大于0.5V时，电流随电压指数增长；反向电压时，电流趋近于 \(-I_s\)。

§17-2 非线性电容和非线性电感

一、非线性电容 - 库伏特性：\(q = f(u)\) 或 \(u = h(q)\)。- 静态电容：\(C = \frac{q}{u}\)。- 动态电容：\(C_d = \frac{dq}{du}\)。二、非线性电感 - 韦安特性：\(\Psi = f(i)\) 或 \(i = h(\Psi)\)。- 静态电感：\(L = \frac{\Psi}{i}\)。- 动态电感：\(L_d = \frac{d\Psi}{di}\)。三、磁滞现象 铁磁材料的 \(\Psi-i\) 特性呈回线形状：磁化曲线不是单值的，与磁化历史有关。磁滞回线包围的面积代表每周的磁滞损耗。

§17-3 非线性电路的方程

特点：基尔霍夫定律仍然适用。元件特性是非线性的，导致电路方程为非线性代数方程或非线性微分方程。解可能不唯一，可能有多个工作点。示例：\(5u^2 + 6u - 8 = 0 \quad \Rightarrow \quad u = 0.8\text{V} \text{ 或 } u = -2\text{V}\)。

§17-4 小信号分析法——核心方法

适用条件：电路中有直流偏置 \(U_0\) 和小信号 \(u_s(t)\)，且 \(|u_s(t)| \ll U_0\)。工作点限制在非线性特性的局部线性区域内。分析步骤：1. 求静态工作点Q：令 \(u_s=0\)，解非线性方程，得 \(U_Q, I_Q\)。2. 计算动态参数：在工作点处求动态电导/电阻。\(G_d = \left. \frac{dg(u)}{du} \right|_{u=U_Q}, \quad R_d = \frac{1}{G_d}\)。3. 作小信号等效电路：非线性元件用动态电阻 \(R_d\) 替代。直流电源置零（电压源短路，电流源开路）。小信号源保留。4. 求解小信号响应：在线性电路中计算。5. 叠加：总响应 = 直流分量 + 小信号分量。\(u(t) = U_Q + u_1(t), \quad i(t) = I_Q + i_1(t)\)。物理意义：小信号分析法将非线性问题局部线性化。类似于函数在一点处的泰勒展开，取线性项。

§17-5 分段线性化方法

基本思想：将非线性伏安特性用若干条直线段近似。在每个线性段内，非线性元件可用线性元件（电阻、电压源）等效。分别计算每个线性段，然后根据工作点所在区域确定有效解。理想二极管模型：正向导通：短路（\(R=0\)，\(u=0\)）；反向截止：开路（\(i=0\)）。隧道二极管的折线近似：用三条不同斜率的直线段近似其伏安特性（正阻区、负阻区、正阻区）。

§17-6 工作在非线性范围的运算放大器

运放的非线性特性：线性区：\(u_d = 0\)，\(-U_{sat} < u_o < U_{sat}\)；正饱和区：\(u_o = U_{sat}\)，\(u_d > 0\)；负饱和区：\(u_o = -U_{sat}\)，\(u_d < 0\)。应用：比较器：输入电压与参考电压比较，输出为高/低电平。滞回比较器（施密特触发器）：引入正反馈，具有抗干扰能力。方波发生器：利用电容充放电和运放饱和特性产生方波。

§17-7 二阶非线性电路的状态平面

状态方程：\(\frac{dx_1}{dt} = f_1(x_1, x_2), \quad \frac{dx_2}{dt} = f_2(x_1, x_2)\)。相平面：以 \(x_1\) 为横坐标、\(x_2\) 为纵坐标的平面。每个点 \((x_1, x_2)\) 代表电路的一个状态。随时间变化的轨迹称为相轨迹。平衡点：满足 \(f_1=0\) 和 \(f_2=0\) 的点。在平衡点处，状态不再变化。RLC串联电路的相图：欠阻尼（\(R<2\sqrt{L/C}\)）：相轨迹为向内螺旋线，收敛于原点。过阻尼（\(R>2\sqrt{L/C}\)）：相轨迹为单调曲线，收敛于原点。无损耗（\(R=0\)）：相轨迹为椭圆，等幅振荡。

§17-8 非线性振荡电路——范德坡电路

电路结构：线性电感 \(L\)、线性电容 \(C\)、非线性电阻（伏安特性 \(i_R = f(u_R)\)）。范德坡方程：\(\frac{d^2x}{dt^2} - \epsilon(1-x^2)\frac{dx}{dt} + x = 0\)。重要现象——极限环：在相平面上，存在一条孤立的闭合曲线（极限环）。无论初始条件在环内还是环外，相轨迹最终都趋向于极限环。这解释了自激振荡现象：振荡的振幅和频率由电路参数决定，与初始条件无关。

§17-9 混沌电路简介

混沌的特征：对初始条件极端敏感：初始值的微小差异会导致完全不同的长期行为（蝴蝶效应）。非周期性：永不重复的复杂运动。有界性：运动被限制在一定范围内。蔡氏电路：最简单的混沌电路之一：两个电容、一个电感、一个线性电阻、一个非线性电阻。状态方程为三阶非线性自治微分方程组。双涡卷吸引子：相轨迹呈现两个“涡卷”形状，永不重复。通向混沌的路径——倍周期分叉：随着参数变化，系统响应从周期1 → 周期2 → 周期4 → 周期8 → ... → 混沌。

§17-10 人工神经元电路

人工神经元模型：输入：多个输入信号 \(x_j\)，每个乘以权重 \(w_j\)。求和：\(\sum w_j x_j + \theta\)（\(\theta\) 为阈值）。激活函数：通常为S形函数（如 sigmoid 函数）。输出：\(y = f(\sum w_j x_j + \theta)\)。Hopfield神经网络：用运放实现神经元。每个神经元的输出通过电阻连接到其他神经元的输入。网络的能量函数单调下降，最终收敛到稳定状态。可用于解决优化问题（如旅行商问题）。

第十八章 均匀传输线

本章从集总参数走向分布参数，研究信号沿传输线的传播特性。核心是理解“长线”与“短线”的区别，以及行波、反射、驻波等概念。

§18-1 分布参数电路

一、为什么需要分布参数模型？ 在集总参数电路中，我们假设：元件间的连接线是理想导线（无电阻、无电感、无电容）。所有电磁过程都集中在元件内部。但实际情况并非如此：导线有电阻 → 沿线有电压降。电流产生磁场 → 沿线有电感。导线间有电压 → 沿线有电容和漏电导。二、长线与短线的区分 - 判断标准：线长 \(l\) 与工作波长 \(\lambda\) 的比较。- 短线：\(l \ll \lambda\)，可用集总参数模型。- 长线：\(l\) 与 \(\lambda\) 可比，必须用分布参数模型。示例：工频50Hz：\(\lambda = 6000\text{km}\)，几百公里的输电线仍是“短线”。高频300MHz：\(\lambda = 1\text{m}\)，几十厘米的导线就是“长线”。三、分布参数的含义 单位长度的参数：\(R_0\)：单位长度电阻（\(\Omega/\text{m}\)），\(L_0\)：单位长度电感（\(\text{H}/\text{m}\)），\(C_0\)：单位长度电容（\(\text{F}/\text{m}\)），\(G_0\)：单位长度电导（\(\text{S}/\text{m}\)）。

§18-2 均匀传输线及其方程

一、均匀传输线的定义 传输线的原参数 \(R_0, L_0, C_0, G_0\) 沿线处处相等。常见类型：平行双线、同轴电缆、双芯电缆等。二、传输线方程（电报方程） 从微元 \(dx\) 分析，由KCL和KVL得：\(-\frac{\partial u}{\partial x} = R_0 i + L_0 \frac{\partial i}{\partial t}\)，\(-\frac{\partial i}{\partial x} = G_0 u + C_0 \frac{\partial u}{\partial t}\)。这是一组偏微分方程，描述了电压和电流随时间和空间的变化。

§18-3 均匀传输线方程的正弦稳态解

一、相量形式的传输线方程 \(-\frac{d\dot{U}}{dx} = Z_0 \dot{I}, \quad -\frac{d\dot{I}}{dx} = Y_0 \dot{U}\)，其中 \(Z_0 = R_0 + j\omega L_0\)，\(Y_0 = G_0 + j\omega C_0\)。二、波动方程 对上述方程求导得：\(\frac{d^2\dot{U}}{dx^2} = \gamma^2 \dot{U}, \quad \frac{d^2\dot{I}}{dx^2} = \gamma^2 \dot{I}\)，其中 \(\gamma = \sqrt{Z_0 Y_0} = \alpha + j\beta\) 称为传播常数。三、通解 \(\dot{U}(x) = A_1 e^{-\gamma x} + A_2 e^{\gamma x}\)，\(\dot{I}(x) = \frac{1}{Z_c} (A_1 e^{-\gamma x} - A_2 e^{\gamma x})\)。- 特性阻抗：\(Z_c = \sqrt{Z_0/Y_0}\)，单位 \(\Omega\)。- 传播常数 \(\gamma = \alpha + j\beta\)：\(\alpha\)：衰减常数（Np/m），表示行波幅度的衰减。\(\beta\)：相位常数（rad/m），表示行波相位的变化。四、入射波与反射波 \(A_1 e^{-\gamma x}\)：向 \(+x\) 方向传播的入射波（从始端向终端）。\(A_2 e^{\gamma x}\)：向 \(-x\) 方向传播的反射波（从终端向始端）。

§18-4 均匀传输线的原参数和副参数

一、副参数 - 传播常数 \(\gamma = \sqrt{(R_0 + j\omega L_0)(G_0 + j\omega C_0)}\)- 特性阻抗 \(Z_c = \sqrt{\frac{R_0 + j\omega L_0}{G_0 + j\omega C_0}}\)二、衰减常数和相位常数的计算 \(\alpha = \sqrt{\frac{1}{2} \left[ \sqrt{(R_0^2 + \omega^2 L_0^2)(G_0^2 + \omega^2 C_0^2)} + (R_0 G_0 - \omega^2 L_0 C_0) \right]}\)，\(\beta = \sqrt{\frac{1}{2} \left[ \sqrt{(R_0^2 + \omega^2 L_0^2)(G_0^2 + \omega^2 C_0^2)} - (R_0 G_0 - \omega^2 L_0 C_0) \right]}\)。三、无畸变条件 - 条件：\(\frac{R_0}{L_0} = \frac{G_0}{C_0}\)- 结果：\(\alpha = \sqrt{R_0 G_0}\)（与频率无关），\(\beta = \omega \sqrt{L_0 C_0}\)（与频率成正比）。- 意义：不同频率的信号衰减相同、传播速度相同，波形不畸变。

§18-5 无损耗传输线

一、无损耗条件 \(R_0 = 0, \quad G_0 = 0\)二、参数简化 \(\gamma = j\omega \sqrt{L_0 C_0} = j\beta\)，\(\alpha = 0\)（无衰减），\(Z_c = \sqrt{\frac{L_0}{C_0}}\)（纯电阻，与频率无关），相速度：\(v_p = \frac{\omega}{\beta} = \frac{1}{\sqrt{L_0 C_0}}\)三、终端开路时的驻波 - 电压：\(u(x,t) = \sqrt{2} U_2 \cos(\beta x) \cos(\omega t)\)（注意：这里 \(x\) 是距终端的距离）- 电流：\(i(x,t) = \sqrt{2} \frac{U_2}{Z_c} \sin(\beta x) \sin(\omega t)\)特点：电压波腹：\(\cos(\beta x) = \pm 1\) 处，即 \(x = k\frac{\lambda}{2}\)（\(k=0,1,2,\dots\)）；电压波节：\(\cos(\beta x) = 0\) 处，即 \(x = (2k+1)\frac{\lambda}{4}\)；电压和电流在空间上相差 \(\lambda/4\)（相位差 \(90^\circ\)）。四、终端短路时的驻波 - 电压：\(u(x,t) = \sqrt{2} Z_c I_2 \sin(\beta x) \cos(\omega t)\)- 电流：\(i(x,t) = \sqrt{2} I_2 \cos(\beta x) \sin(\omega t)\)特点：与开路情况相比，电压和电流的波腹、波节位置互换（移动 \(\lambda/4\)）。五、输入阻抗 - 开路时：\(Z_{in} = -j Z_c \cot(\beta l)\)- 短路时：\(Z_{in} = j Z_c \tan(\beta l)\)应用：长度 \(<\lambda/4\) 的开路线 → 等效电容；长度 \(<\lambda/4\) 的短路线 → 等效电感；长度 \(=\lambda/4\) 的线 → 阻抗变换器：\(Z_{in} = Z_c^2 / Z_L\)。

§18-6 无损耗线方程的通解（时域）

无损耗线的时域方程：\(\frac{\partial u}{\partial x} = -L_0 \frac{\partial i}{\partial t}, \quad \frac{\partial i}{\partial x} = -C_0 \frac{\partial u}{\partial t}\)。通解：\(u(x,t) = u_1\left(t - \frac{x}{v}\right) + u_2\left(t + \frac{x}{v}\right)\)，\(i(x,t) = \frac{1}{Z_c} \left[ u_1\left(t - \frac{x}{v}\right) - u_2\left(t + \frac{x}{v}\right) \right]\)，其中 \(v = 1/\sqrt{L_0 C_0}\)：波速，\(u_1(t - x/v)\)：向 \(+x\) 方向传播的正向行波，\(u_2(t + x/v)\)：向 \(-x\) 方向传播的反向行波。

§18-7 无损耗线的波过程

一、波的产生与传播 当开关闭合瞬间：电压波以速度 \(v\) 沿传输线传播。同时伴随电流波：\(i = u / Z_c\)。能量以电磁波形式传输。二、波的反射 - 终端开路：电流反射波与入射波大小相等、方向相反（使总电流为零）；电压反射波与入射波大小相等、方向相同（使总电压加倍）。- 终端短路：电压反射波与入射波大小相等、方向相反；电流反射波与入射波大小相等、方向相同。三、多次反射过程（以开路线接直流为例） 1. \(0 < t < l/v\)：入射波传播，沿线电压 \(U_0\)，电流 \(I_0 = U_0/Z_c\)。2. \(t = l/v\)：波到达终端，发生反射。电压变为 \(2U_0\)，电流变为 0。3. \(l/v < t < 2l/v\)：反射波向始端传播。4. \(t = 2l/v\)：波到达始端，再次反射。电压变为 \(U_0\)，电流变为 \(-I_0\)。5. 如此循环，周期 \(T = 4l/v\)。四、匹配 - 当负载电阻 \(R_L = Z_c\) 时，无反射波（匹配）。- 能量全部被负载吸收，传输效率最高。- 这是通信线路追求的理想状态。
